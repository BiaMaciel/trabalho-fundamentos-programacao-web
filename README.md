# Projeto: Portfólio Pessoal - Fundamentos da Programação Web

Este projeto foi desenvolvido como parte da disciplina de **Fundamentos da Programação Web**. O objetivo principal foi aplicar conceitos iniciais de estruturação e estilização de interfaces, criando um portfólio funcional e organizado.

---

## 🚀 Sobre o Projeto

O portfólio foi construído utilizando tecnologias nativas da web para garantir uma base sólida em semântica e design. A navegação é feita de forma intuitiva através de um menu de botões que direcionam o usuário para páginas específicas de conteúdo.

### 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica de todas as páginas.
* **CSS3**: Estilização visual, layout e design dos botões de navegação.

---

## 📁 Estrutura de Navegação

O projeto é composto por quatro páginas principais, cada uma acessível pelo menu de navegação:

* **Home**: Página de aterrissagem com apresentação inicial.
* **Sobre Mim**: Seção dedicada à trajetória pessoal e profissional.
* **Formação**: Detalhamento da vida acadêmica e cursos realizados.
* **Contato**: Canais de comunicação e links externos.
